# How to run crawler

## 1. Download crawler

```sh
git clone repo-name name-folder
cd name-folder
```

## 2. Fetch packages

```sh
npm install
```

## 3. Run crawler

```
npm run start:dev -- --start
```

> Parameter **--start** starts the crawler without wait for cron start.

# Required

-   node v12.16.2
-   npm 6.14.4
